## Como publicar no GitHub Pages

1. Substitua os arquivos antigos pelos arquivos desta pasta.
2. Mantenha a estrutura:

```text
index.html
README.md
audio/
  acerto.ogg
  erro.ogg
```

3. Rode:

```bash
git add .
git commit -m "Corrige audios e gabaritos do simulado PC PR 2026"
git push
```

4. No navegador, use Ctrl + F5 ou abra em janela anônima para evitar cache antigo.


## Versão responsiva para celular e tablet

Esta versão reduz o cabeçalho, deixa o cronômetro e o som em formato compacto, transforma o mapa de disciplinas em painel lateral no celular/tablet e amplia a área destinada às questões.

- No computador, o mapa fica em uma coluna estreita.
- No celular e tablet, use o botão **☰** para abrir o mapa de questões.
- O cabeçalho mostra apenas o essencial.
- Questões e alternativas ocupam a maior parte da tela.
- Feedback e explicação foram compactados sem perder legibilidade.
